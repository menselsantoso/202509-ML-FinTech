Machine Learning and FinTech
Bridging AI, Statistics, and Finance through Learning by Doing

About This Course
This course at NYCU (National Yang Ming Chiao Tung University, 國立陽明交通大學) embraces the rapid development of AI while remaining firmly rooted in the principles of statistical learning. It provides students with both conceptual foundations and hands-on experience in applying machine learning methods to financial technology problems.

This repository serves pedagogical purposes, offering a collaborative platform where students can share ideas, exchange code, and sharpen their knowledge in Machine Learning and FinTech.
